Thanks for using the Patreon Downloader

What's up

This extension is a labour of love for me. I programmed this it to solve a problem that I had. Archiving the photos from patreon creators. I needed a way to scrape slideshows from the responsive html which turned out to be a much more difficult task than I originally thought. I released this plugin on the chrome web store in case it was useful for anyone else. It's an open source project. Feel free to contribute or report bugs at to at github.

What this extension does.

  • Downloads files linked in posts
  • Downloads the post header image
  • Attempts to mimic a user to scrape and download slideshows
  • Saves the text and comments from a post

What this extension DOES NOT do

  • Download streaming video
  • Download content that is above the level that you've paid for
  • Work on patreon in languages other than english
